Polynomial Calculator

The Polynomial Calculator is a C-based application designed to perform arithmetic operations on polynomials using linked lists. This project aims to showcase the practical implementation of linked lists in solving polynomial addition, subtraction, and multiplication problems. The calculator efficiently manages and manipulates polynomials represented as linked lists, allowing for dynamic memory allocation and deallocation.

Gare Up Garage

Gare Up Garage is a web development project designed to streamline the process of booking car repair appointments online. Built using React.js for the front-end and MongoDB for the back-end, this website not only allows users to schedule appointments conveniently but also maintains a detailed record of past services performed on their vehicles. This ensures a seamless and efficient experience for both the customers and the garage management.
